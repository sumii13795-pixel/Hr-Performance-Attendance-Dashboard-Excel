# Hr-Performance-Attendance-Dashboard-Excel
Excel-based HR Dashboard to track employee performance, attendance, and workforce insights using interactive visuals.

📌 Project Overview
This project is an advanced Excel-based HR Dashboard designed to monitor employee performance and attendance.
It helps HR teams track workforce data, analyze trends, and make better decisions.

🚀 Features
✔ Employee Attendance Tracking
✔ Performance Analysis
✔ Department-wise Insights
✔ Interactive Dashboard with Charts
✔ Automated Calculation

📊 Dashboard Insights
Attendance Overview
Performance Metrics
Department-wise Analysis
Employee Trends

🛠 Tools & Skills Used
Microsoft Excel
Pivot Tables
Charts & Graphs
Data Analysis
HR Analytics
